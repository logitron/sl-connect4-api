> Rails 5 API for Connect 4 Application

  <img src="https://github.com/logitron/sl-connect4-api/blob/master/connect4.jpg" alt="Connect 4" />

<br />

* **Ruby Version**: 2.4.2
* **Database**: Postgresql

## Setup
```
// Create Databases
rake db:create

// Run Database Migrations
rake db:migrate

// Run the Server
rails s
```

## Testing
```
// Simply run rake
rake
```
